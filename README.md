**Welcome To Growth Function Fight Club**

This is a game I made for players to learn about growth functions in a fun way. I made this game almost entirely with Ai, more specifically a mix of pro and free searches with Perplexity using the GPT-5.1 model. I also used deepai.org for the character images combined with remove.bg to remove the background from those characters. For the music I used pixabay.com for free sounds I picked.

**Features**

This game features growth functions, character models, game win/loss, betting, graphs, tug of war like display, fun immersive sounds, and last but not least, an Ai helper in the form of a gemini api to help you with why you got certain answers correct or incorrect.

**How To Run Locally**

You're going to want npm and node installed. Then it's as simple as typing npm start while in the main directory of this game. Visit your localhost at port 5000 and you will be able to play the game locally
